The Rates component uses ideas from:

  • https://www.apollographql.com/docs/angular/basics/setup.html
  • https://github.com/patricknazar/angular-lazy-loading-apollo-client
  • https://o5x5jzoo7z.sse.codesandbox.io/graphql

Make sure to load the codesandbox (for the GraphQL API) in another tab while testing, otherwise you'll see errors in devtools and the rates will not load.

The mock rates will be rendered on the server (JavaScript not required on the client) if angular-x-postpress is loaded using:

npm run build:ssr && npm run serve:ssr
