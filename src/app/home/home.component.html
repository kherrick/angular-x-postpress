About x-postpress

x-postpress is a Web Component that fetches WordPress posts from the REST API based on attributes set on the element. It's built using LitElement and hosted on npm. To try it out, add a script tag to your document:

<script
  type="module"
  src="https://unpkg.com/x-postpress"
>
</script>

Then declare the x-postpress element with the apiHost attribute set to the appropriate WordPress installation:

<x-postpress
  apiHost="https://content.example.com"
>
</x-postpress>
